# Web Automation + Excel Report

Automação em Python que acessa um site, coleta dados automaticamente e gera um relatório em Excel.

## O que faz

- Faz login em um sistema web  
- Coleta produtos e preços  
- Organiza os dados  
- Gera um arquivo Excel automaticamente  

## Tecnologias

- Python  
- Selenium  
- Pandas  

## Como usar

1. Instale as dependências:
pip install selenium pandas openpyxl

2. Execute:
python main.py

3. Informe quantos produtos deseja coletar

## Resultado

Gera um arquivo `produtos.xlsx` com os dados coletados.

## Objetivo

Reduzir tarefas manuais e automatizar coleta de dados em sistemas web.
